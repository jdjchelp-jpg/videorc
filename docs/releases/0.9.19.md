# Videorc 0.9.19 Beta 1

Videorc 0.9.19 ships the live layout switching fix for active recording and
streaming sessions, plus the manual Twitch stream comments fix.

## Scope

- **Live layout switching while active** (`live_layout.rs`, `use-studio.tsx`) -
  scene changes can commit while recording or streaming even when a valid
  screen/window source has not visibly changed. This specifically covers the
  "no fresh frames within 5s" blocker reported during recording.
- **Recording-studio smoke coverage** (`smoke-live-layout-switch-recording-app.mjs`) -
  the active-session layout smoke now proves apply_live switches during both a
  recording session and a record+stream session, then inspects finished media
  artifacts instead of relying on button/state checks.
- **Manual Twitch stream comments** (`twitch_chat.rs`, `live_chat.rs`,
  `live-chat-view.ts`) - manual Twitch streams can attach the Twitch live chat
  connector when the connected account has the chat scope, and the comments UI
  surfaces actionable provider state.
