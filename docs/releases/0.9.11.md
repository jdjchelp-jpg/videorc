# Videorc 0.9.11 Beta 1

Videorc 0.9.11 is the **camera shape** release — and the first release
delivered by the automatic launch check that 0.9.10 shipped.

## Scope

- **Camera shape + aspect** (C1–C6, plan in vault
  `2026-07-06 - Videorc Camera Shape And Aspect Plan.md`) — `CameraShape`
  grows Rounded (rounded SQUARE by default: picking it snaps
  `cameraAspect` to square; corner-radius slider 0–50% of the shorter
  side); new `cameraAspect` = source | square | portrait (3:4 center-crop
  via the default Fill fit — the vertical-camera framing). The mask is one
  shared `SourceMask` enum consumed by all three render paths (CPU SDF,
  Metal shader in aspect-normalized space, FFmpeg geq) with agreement
  pinned by CPU corner/edge tests, an on-device GPU pixel test, and a
  golden filter expression. Scene stage draws the true corner radius.
  Old configs deserialize to defaults on both sides.
- **Studio health** — recording with the preview window closed no longer
  reports "Blocked" (648740f4, other session).
